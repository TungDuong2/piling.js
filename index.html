Piling.js

Piling.js is a library for visual piling of images, matrices, scatterplots, and other types of data visualizations.
(Data from the Lin et al., 2014) (Data from Lekschas et al., 2018) (Data from Johns Hopkins University) (Data from Google Quickdraw Dataset) (Data from NASA GISTEMP) (Data from Codeluppi et al., 2018 based on Viv ) (Data from World Bank Open Data) (From Brian B, 2014 inspired by Bach et al., 2015 )
